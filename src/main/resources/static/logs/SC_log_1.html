我为什么要建立这个网站？

时间戳：2021-1-1


建立个人网站这个想法始于大二的Web前端课程，当时第一个念头就是“如果能凭借所学的知识建立一个属于自己的网站那一定很酷！”。

后来细细想来也确实是一件很有意义的事情，也决定要首先写下一篇“建站志”作为第一篇文章——一个起点：这个网站之于我的意义，希望能够“不忘初心”（尽管这个词已经被各方用烂了，但我记住这个词是在此之前的）。

首先，我希望这是一块属于我的互联网自留地。

一个自由地可以写下我所思所想的地方，记录以及总结输出。它不需要被太多人看到，最重要的是对我，其次是对在意我的人——可以有一个更深层次地了解我的窗口。空间、朋友圈、微博甚至CSDN，都太世俗、太浮躁、太喧嚣了，尽管会有更多的连接，但我拒绝在这些地方写东西，我希望这个网站可以代替并超越他们。

其次，这是我锻炼自己专业知识的“实验室”。

毕竟我一直认为“计算机”就是我想要的，同时作为一名“软件工程专业”的学生，“搭建一个网站”、学习并实践这个过程中的种种技术无疑是一个非常有意义的锻炼！它能让我主动地去学习并学以致用，提升我的专业能力！

再者，它拥有无限可能。

可以确定的是——他首先会是一个”博客“，还能是什么？还可以是一份”个人简历“，我想校招的时候会是个加分项；还能是——一个”个人网盘“，云储存我的文件，”书签托管“应该也和这个差不多；还可以是……”自建VPN“，虽然我没打算这么做，我觉得没必要；还可以是一个盗版漫画网站 / 盗版视频网站，我可以学习”爬虫“并靠广告盈利；还可以是一个”电商网站“，哈哈朝商业的方向偏了……总之，正如我所说的，我觉得它会有无限可能。

那么谈谈目前的现状和未来的发展规划吧

毕竟今天是2021年的第一天，立下一个”新年flag“似乎特别符合时宜。

目前，这是阿里云上部署的，并且采用Wordpress建站程序搭建的。”小白式“的搭建方法，好吧其实用起来还不错，看起来也不错。但，作为一个未来的专业人士我其实是非常抗拒这样做的，专业人士当然要自己一步步来啦。然而……无奈的是实力不允许，目前也就才入个门而已，”要一步步自己搭建一个有趣的网站“无疑是好高骛远说大话了。

零零碎碎了解到的，大概有”LAMP“即Linux + Apache + MySQL + PHP路线（也是我接下来打算走的路线）和JAVA Web + Tomcat路线（这个后面学校要教一部分）。但因为降级转专业提前毕业的原因，下学期课很多还要学C++，”LAMP“里面一个也没学过（虽然感觉不会花很多时间每一项，毕竟不难要求也不高），那……这么看来的话合理的预期是下一个暑期结束前完成这一设想吧！

再想远一点，后面应该有可能（这么说是非常纠结的语气）会换服务器，换成国外的，毕竟众所周知国内有审查，还是不够”自由“，不过这也是后话了。
